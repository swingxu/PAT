# PAT

PAT 网站的题目是按照每次考试的遍题顺序罗列的，比较适合备考后期模考，但不适合前期系统性的针对知识点进行训练以及后期弱点巩固。
备考的时候按照知识点重新整理了下PAT考题（陆续更新中），需要的同学可以瞅一眼～

初学算法，采用的是算法笔记中 <C+STL> 的编码方式，不一定是最优解，不足之处请多多包涵。
